# Market Place

A clean, scalable SwiftUI marketplace app with listing browsing, favorites, detail views, and offline listing creation. Built with a modular architecture using three local SPM packages (`Models`, `Networking`, `Core`) and dependency injection via a `DependencyContainer`.

Demonstrates a clean SwiftUI modular setup with dependency injection and testable layers suitable for scaling features like the marketplace listing view.
